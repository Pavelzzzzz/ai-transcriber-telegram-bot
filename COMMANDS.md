# 🤖 **Команды, поддерживаемые AI Транскрибатором**

## 📋 **Основные команды для всех пользователей:**

### 🔧 **Управление ботом**
- **`/start`** - Запуск бота и приветствие
- **`/help`** - Показать подробную справку
- **`/mode`** - Выбор режима работы бота
- **`/status`** - Показать текущий статус и активность
- **`/profile`** - Профиль пользователя с статистикой

### 🎯 **Режимы работы:**
- **📸 "Изображение → Текст"** - OCR распознавание текста на фото
- **🎤 "Аудио → Текст"** - Транскрибация голосовых сообщений
- **🔊 "Текст → Аудио"** - Преобразование текста в речь (TTS)

---

## 📋 **Команды в режиме "Изображение → Текст":**
- **Отправка фотографии** - Просто отправьте фото с текстом
- **Автоматическое распознавание** - Бот автоматически обработает изображение
- **Результат** - Распознанный текст в ответном сообщении

---

## 🎯 **Команды в режиме "Аудио → Текст":**
- **Отправка голосового** - Запишите голосовое сообщение
- **Поддерживаемые форматы:**
  - Голосовые сообщения Telegram
  - Аудиофайлы (WAV, OGG, MP3, M4A)
- **Автоматическая транскрипция** - Whisper AI обработка
- **Результат** - Распознанный текст в ответном сообщении

---

## 🎯 **Команды в режиме "Текст → Аудио":**
- **Отправка текста** - Напишите текстовое сообщение
- **Поддержка длинных сообщений** до 5000 символов
- **Голосовое озвучивание** - Google TTS (Text-to-Speech)
- **Языки:** Русский и английский
- **Результат** - Аудиофайл в ответном сообщении

---

## 🎯 **Дополнительные функции:**

### 📊 **Команда `/history`:**
- Показывает историю транскрипций пользователя
- **Фильтрация по типам:**
  - 📸 Изображение → Текст
  - 🎤 Аудио → Текст  
  - 🔊 Текст → Аудио
- **Статус каждой транскрипции:** ✅ завершено, ❌ ошибка, 🔄 в процессе

### 📈 **Команда `/stats`:** (администратор)
- Общая статистика использования бота
- Количество транскрипций по типам
- Статистика пользователей

### 🎛 **Команда `/mode`:**
- **Интерактивное меню** с выбором режима
- **Быстрое переключение** между режимами
- **Отображение текущего режима**

---

## 🏛️ **Административные команды:**

### 🔒 **Команда `/admin`:** (администраторы)
- **Статистика** - Общая статистика использования
- **Пользователи** - Управление пользователями
- **Блокировки** - Блокировка/разблокировка пользователей
- **Логи** - Просмотр логов системы

### 📊 **`/stats`** - Статистика
- Всего транскрипций в системе
- Транскрипции по типам
- Успешные/неуспешные
- Среднее время обработки

### 👥 **`/users`** - Управление пользователями
- Список всех пользователей
- Фильтрация по статусу (активные, заблокированные)
- Детальная информация о каждом пользователе

### 🚫 **`/block <id>`** - Блокировка
- Блокировка пользователя по ID
- Указание причины блокировки
- Логирование действия

### 🔄 **`/unblock <id>`** - Разблокировка
- Снятие блокировки с пользователя
- Логирование действия

## 🔧 **Дополнительные возможности:**

### ✨ **Автоматическое распознавание:**
- **Smart OCR** - tesseract с улучшениями
- **Multiple languages** - русский, английский
- **Форматирование** - сохранение структуры текста

### 🎤 **Мощная транскрипция:**
- **Whisper AI** - state-of-the-art модель
- **Множественные модели** - tiny, base, small, medium, large
- **Автоопределение языка** - поддержка 99+ языков

### 🔊 **Качественное озвучивание:**
- **Google TTS** - естественный голос
- **Настраиваемая скорость** - медленная/нормальная/быстрая
- **Разные голоса** (только для английского)
- **Длинные тексты** - до 5000 символов

---

## 📋 **Обработка ошибок:**
- ✅ **Graceful degradation** - при ошибках понятные сообщения
- 🔍 **Детальное логирование** - все ошибки записываются
- 📊 **Уведомления** - администраторы уведомлены о проблемах
- 🔄 **Автоповторы** - автоматические повторные попытки

---

## 🎯 **Интеграция с Telegram:**
- **Photo handler** - автоматическая обработка фотографий
- **Voice handler** - автоматическая обработка голосовых
- **Text handler** - автоматическая обработка текстовых сообщений
- **Command handlers** - обработка slash команд
- **Callback handler** - обработка inline кнопок
