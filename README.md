# 🤖 AI Транскрибатор - Telegram Бот

**Интеллектуальный Telegram-бот для преобразования контента между различными форматами с помощью AI технологий**

## 📋 Описание проекта

AI Транскрибатор - это современный Telegram-бот, который использует передовые технологии искусственного интеллекта для многофункциональной обработки контента. Бот поддерживает 4 основных режима работы: распознавание текста на изображениях, преобразование речи в текст, синтез речи из текста, и интеллектуальную обработку текста.

### 🎯 Основные возможности

- 📸 **Распознавание текста (OCR)** - Извлечение текста из изображений
- 🎤 **Распознавание речи** - Преобразование голосовых сообщений в текст с помощью Whisper AI
- 🔊 **Синтез речи (TTS)** - Преобразование текста в качественное аудио
- 💬 **Интеллектуальная обработка текста** - Анализ, исправление и улучшение текста
- 🌍 **Мультиязычность** - Поддержка русского, английского и других языков
- ⚡ **Быстрая обработка** - Оптимизированная обработка всех типов контента
- 📱 **Удобный интерфейс** - Интуитивное управление через Telegram
- 👥 **Пользовательские роли** - Разграничение доступа (Пользователь/Администратор)
- 📊 **Статистика и аналитика** - Детальная информация об использовании

## 🎮 Использование

### 📋 Основные команды

#### 👤 Команды для всех пользователей
- `/start` - 🚀 Запуск бота и приветственное сообщение
- `/help` - 📖 Подробная помощь и инструкции
- `/mode` - 🎛️ Выбор режима работы
- `/status` - 📊 Проверка статуса бота

#### 🔐 Команды для администраторов
- `/admin` - 🔛 Панель администратора
- `/stats` - 📈 Статистика использования
- `/users` - 👥 Список пользователей
- `/logs` - 📝 Системные логи

### 🔄 Режимы работы

#### 📸 Фото → Текст (OCR)
1. **Отправьте изображение** (JPG, PNG, WEBP)
2. **Бот распознает текст** с помощью OCR
3. **Получите распознанный текст** в markdown формате

#### 🎤 Голос → Текст (Whisper)
1. **Отправьте голосовое сообщение**
2. **Бот транскрибирует речь** с помощью Whisper AI
3. **Получите текст** распознанной речи

#### 📝 Текст → Голос (TTS)
1. **Отправьте текстовое сообщение**
2. **Бот синтезирует речь** из текста
3. **Получите аудиофайл** с озвученным текстом

#### 💬 Текст → Ответ (AI)
1. **Отправьте текст**
2. **Бот анализирует и улучшает текст**
3. **Получите исправленный текст** с рекомендациями

### 💡 Советы для лучших результатов

#### 📸 Для изображений:
- Используйте четкий, контрастный текст
- Убедитесь, что текст хорошо освещен
- Предпочтительнее изображения высокого разрешения
- Текст должен быть горизонтальным и читаемым

#### 🎤 Для голосовых сообщений:
- Говорите четко и отчетливо
- Избегайте сильного фонового шума
- Используйте встроенный рекордер Telegram
- Поддерживаются русский и английский языки

#### 📝 Для текста:
- Текст может быть на любом языке
- Поддерживаются длинные сообщения
- Бот исправит ошибки и предложит улучшения

## ⚙️ Конфигурация

### 🔧 Переменные окружения (.env)

```env
# 🤖 Telegram Bot Configuration
TELEGRAM_BOT_TOKEN=your_bot_token_here
ADMIN_USERNAMES=admin1,admin2  # Имена администраторов

# 🧠 AI Models Configuration
WHISPER_MODEL=tiny              # tiny/base/small/medium/large
WHISPER_DEVICE=cpu             # cpu/cuda
LOG_LEVEL=INFO                 # DEBUG/INFO/WARNING/ERROR

# 📁 File and Size Settings
MAX_FILE_SIZE_MB=20            # Максимальный размер файла
RATE_LIMIT_PER_MINUTE=10       # Лимит запросов в минуту

# 🛡️ Security Settings
CORS_ORIGINS=http://localhost:3000
```

### 🎛️ Доступные модели Whisper

- `tiny` - ⚡ Самая быстрая, базовое качество (~32MB)
- `base` - ⚖️ Сбалансированная скорость и качество (~142MB)
- `small` - 🎯 Улучшенное качество (~466MB)
- `medium` - 🔥 Высокое качество (~1.5GB)
- `large` - 🚀 Максимальное качество (~2.9GB)

## 🐛 Устранение проблем

### 🔧 Частые проблемы и решения

#### 🐳 Docker проблемы:
```bash
# ❌ Ошибка: Cannot write to logs directory
# ✅ Решение:
sudo chown -R $USER:$USER logs/

# ❌ Ошибка: Port already in use
# ✅ Решение:
docker-compose down
docker-compose up -d
```

#### 🤖 Проблемы с ботом:
```bash
# ❌ Ошибка: Bot token not found
# ✅ Решение:
echo "TELEGRAM_BOT_TOKEN=your_token" >> .env

# ❌ Ошибка: Module not found
# ✅ Решение:
pip install -r requirements.txt
```

#### 🧠 AI/ML проблемы:
```bash
# ❌ Ошибка: Whisper not working
# ✅ Решение:
pip install --upgrade openai-whisper

# ❌ Ошибка: Tesseract not found
# ✅ Решение:
sudo apt-get install tesseract-ocr tesseract-ocr-rus
```
